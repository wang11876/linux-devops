## 1.

正则模式

```
/user(/|$)(.*)
```

可以匹配到如下内容

```
/user
/user/
/user/general
```

该正则表达式中存在两对括号, 其中`(/|$)`用于匹配`/user/`和`/user`两种情况, `|`表示"或"的情况.

如果`/user`已经被`$`捕获, 表示已到结尾, 后面的`(.*)`就没啥用了, 比如上面的正则就无法匹配`/user123`.

## 2.

正则模式

```
([^:]+)(?::\d+)?
```

可以匹配到IP:port

```
127.0.0.1:8080
```
